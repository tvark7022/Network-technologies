Сервер:
Создаётся контекст для работы ZeroMQ
Потом сокет, через который мы будем отсылать сообщеня
Далее методом "Bind" привязываем его к ip адресу и порту
Печатаем в консоль, что всё хорошо
Запускаем бесконечный цикл, который постоянно будет просить ввести новое сообщение
Сообщение отслается клиентам
Клиент:
Создаёт контекст, в котором будет работать ZeroMQ
Так же создаёт сокет, через который будет принимать сообщения
И подключается к серверу по ip и порту методом Connect
Далее в бесконечном цикле принимает сообщения в виде объектов ZFrame
Метод ReceiveFrame ждёт новое сообщение. Пока оно не пришло, программа дальше не выполняется
Далее принятое сообщение печатается в консоль
И цикл повторяет эти действия снова
